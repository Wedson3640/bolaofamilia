# Fanpage Bolão Família Online

Landing page em Next.js + Tailwind CSS para o domínio `bolaofamilia.online`.

## Como rodar

```bash
npm install
npm run dev
```

Acesse:

```text
http://localhost:3000
```

## Deploy recomendado

Vercel.

## Imagem

A imagem principal está em:

```text
public/images/fanpage-bolao.png
```

## CTA

O botão principal aponta para:

```text
/cadastro
```

Altere para a rota real do seu sistema quando estiver pronta.
